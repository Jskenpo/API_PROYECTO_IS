# MANEJO DE DATOS (API) DE COSITAMIA

## Introducción
¡Bienvenido a la API de Cositamía! Esta API ha sido diseñada para facilitar el manejo de datos relacionados con un carrito de compras en la aplicación web de cositamía. Con esta API, podrás realizar operaciones esenciales para gestionar los artículos en el carrito, agregar nuevos productos, eliminar elementos, calcular el total de la compra e incluso encriptación de datos.
